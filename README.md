# Draw Pit Dip Sheet Calculator

A professional web-based calculator for determining Stackabox levels in draw pit construction with B125 cover, bedding, and laser datum offset calculations.

## 📋 Features

✅ **User-Definable Inputs**
- Lowest duct level (mm AOD)
- Rotating laser datum height (mm AOD)

✅ **Automatic Calculations**
- Whole number of 150mm Stackabox units
- All component level elevations
- Required depth analysis

✅ **Visual Output**
- Color-coded cross-section diagram
- Real-time laser datum reference line
- Component offset visualization

✅ **DIP Sheet Table**
- All level descriptions
- Absolute elevations (mm AOD)
- Depth below finished level
- **Laser datum offsets** for site layout

✅ **Mobile Friendly**
- Works on desktop, tablet, and Android phone
- Responsive design
- Offline capable

## 📊 DIP Sheet Output

The calculator generates a complete dip sheet showing:
- Finished compound level
- B125 cover top level
- Bedding level
- Lowest duct level
- Each Stackabox unit level
- Sump unit level
- Drain/invert level

**All with calculated offsets from your laser datum for site layout!**

---

## 📝 Notes

- All calculations assume whole number of 150mm Stackabox units (automatically rounded up)
- Laser datum is typically set above the finished compound level
- Negative offsets mean the laser needs to be lowered to that level
- Positive offsets mean the laser is above that level
